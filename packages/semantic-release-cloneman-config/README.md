# `@forsakringskassan/semantic-release-config`

> Försäkringskassan semantic-release config for [Cloneman](https://github.com/Forsakringskassan/clonemans) templates.

## Usage

```bash
npm install --save-dev @forsakringskassan/semantic-release-cloneman-config
```

In `package.json`:

```json
{
    "release": {
        "extends": "@forsakringskassan/semantic-release-cloneman-config"
    }
}
```
